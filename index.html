How javascript Run ?

javascript is not like C, C++ or other compiled based language but an interpreted one.We don't need to use any compiler machine(like C, C++ to compile code) to send the js source code in browser.It is an automaticlly interpreted and run in web browser.Compiler recieves the source codes and optimizes as well as compiles.Finally, after compilation turning into translation then bytecode. Actullay Javascript engine takes the code and parses, converts to machine and finally run.Every Web browser has their own Engine(e.g. V8 engine for chrome). To wrap up, Javascript works : Source Code to Baseline Compiler to bytecode to Interpreter to Binary to CPU

what is the difference between localstorage and sessionstorage?

Differences and similarities of them:
1. They both are web storage API.
2. Local storage stores data but it has no expiration.But Session storage data is expired and it is lost when browser tab is closed.
3.They both stores keys and values in browser by adding setItem() and GetItem() removeItem() and ClearItem()
4. They have a limitation of size about 5 MB.
5. The values are set by only strings.
6. they are also window objects like history and cookies.
7. We can see the storage of them by clicking inspect elements then appication tab.
